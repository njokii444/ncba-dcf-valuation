### NCBA group DCF valuation
This project presents a DCF valuation analysis of NCBA group a leading financial services provider in Kenya. The analysis aims to estimate the intrisic value of the company based on it's projected future cash flos.

## Project overview
**objectives** To evaluate intrisic value and compare it with it's market value usimg DCF method.
**tools used** Microsoft Excel,financial statements and public market data.

## Methodology
Financial data collection
Forecasting free cash flows
Discount rate calculation
Terminal value
Sensitivity analysis

## Key findings
**estimated intrinsic value per share** 
**comparison to market price** 
**investment implicaton** If intrinsic value is greater than market price the stock price is undervalued.
